前端基础学习用。
============
文件目录：
--------
* `case`:小案例。<br>
* `css`:练习用到的CSS文件。<br>
* `html`：练习用到的html文件。
* `image`：练习过程中用到的图片。
* `note`：学习笔记。
* `practice`：一些综合的小练习。
* `psd`:psd的网页文件（ps:以后可能是UI小姐姐给的）
