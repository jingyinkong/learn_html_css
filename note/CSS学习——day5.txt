CSS的复合选择器
	后代选择器(重要)
		元素1 元素2{样式声明}
		需要空格隔开
		元素1是父级，元素2是子级，最终选择的是元素2
		只要是元素1的后代都可以选择
		.nav li a {
			color: pink;
		}
		这样也可以选择类下面的标签
	子选择器
		只能选择最近一级的子元素
		元素1>元素2{样式声明}
		元素1是父级，元素2是子级，最终选择的是元素2
	并集选择器
		选择多组标签定义相同的样式，通常用于集体声明
		元素1，元素2{样式声明}
		用逗号隔开
		语法规范竖着写
		最后一个选择器不需要加逗号
	伪类选择器
		给链接添加效果
		书写最大特点是用：表示
			链接伪类
				a:link(a:any-link)		选择所有未被访问的链接
				a:visited				选择所有已被访问的链接
				a:hover					选择鼠标指针位于其上的链接
				a:active				选择活动链接(鼠标按下未弹起的链接)(一直点着)
				注意事项
					为了保证确定可以生效，请按照LVHA的声明顺序声明：link visited hover active
					(love hate)
					实际需要给a标签单独指定样式
					实际开发中是先写a标签的样式，再去写链接伪类选择器
			结构伪类
			表单伪类
			：focus伪类选择器
				用于选取获得焦点的表单元素
				一般情况用于<input>类表单元素
				input:focus {
					background-color: khaki;
				}
